Descriptiontop

Domain tools is a class that can be used to get information about a domainname. The information about the domain is retrieved from Google, Bing, Yahoo, Alexa and DNS servers.

  • Check if a domainname is free. Supported extensions: be, biz, cn, com, co.uk, edu, eu, in, info,mobi, name, net, nl, nu, org, tv, us, ws, hu, pl, se
  • Get Google and Alexa rank for a domain
  • Indexed pages on Google, Bing and Yahoo
  • Count backlinks with Google and Yahoo
  • Whois information
  • Top keywords for a domain
  • 3 examples included
  • Example of how to show all info for 1 specific domain.

How to install top

Extract all the files form the zip. All the files in the lib folder are the classes you need. First thing to do is including the DomainTools.php file in your script and create an instance of it.

include("lib/DomainTools.php");
$DomainTools = new Base_DomainTools();

Now that we have an instance of DomainTools we can call methods on this object. An example of a method is Rank, to get the Google and Alexa ranks of a domain. For example to get the Ranks of the domainname themeforest.net you use the following code.

$result = $DomainTools->Rank("themeforest.net");

The method Rank, returns an array with keys; google, alexa, alexa_delta. If you want to show the Google pagerank than you add the following line of code.

echo $result['google'];

Methods and parameters top

In the previous section i explainded how you get the Google pagerank for a domainname with the Rank method. In this section i give you an overview of all the methods in the DomainTools class.

  1. Rank: Get the Google and Alexa rank for a domainname
    • Parameter: domainname, for example themeforest.net
    • Return: Array with 3 keys, google, alexa, alexa_delta
    • Code example:
      $result = $DomainTools->Rank("themeforest.net");
echo $result['google'] . "<br />";
echo $result['alexa'] . "<br />";
echo $result['alexa_delta'] . "<br />";
  2. Keywords: Returns a list of top keywords for a domainname
    • Parameter: domainname, for example themeforest.net
    • Return: Array
    • Code example:
      $result = $DomainTools->Keywords("themeforest.net");
echo "<ul>";
foreach($result as $keyword){
echo "<li>" . $keyword . "</li>";
}
echo "</ul>"
  3. Whois: Get whois information from a domainname
    • Parameter: domainname, for example themeforest.net
    • Return: String
    • Code example:
      $result = $DomainTools->Whois("themeforest.net");
echo nl2br($result);
  4. Free: Check if a domainname is free
    • Parameter1: domainstring, for example themeforest. Its important that you leave the extension off!
    • Parameter2: Extension, empty=return the result for all extensions | array=return the result for the extensions in the array | string=return the result for one extension
    • Return: Can be a boolean of an array with keys and booleans depending on how many extensions your checking
    • Code example1:
      $result = $DomainTools- >Free("mytestpage", array("be", "us", "com"));
foreach($result as $extension => $status){
$string_status = ($status) ? "Free" : "Used";
echo ".<strong>" . $extension . "</strong>: " . $string_status . "<br />";
}
    • Code example2: 
      $result = $DomainTools->Free("mytestpage", "be");
$string_status = ($result) ? "Free" : "Used";
echo "mytestpage.be is " . $string_status;
  5. IndexedPages: Get number of indexed pages for a domainname in Google, Bing and Yahoo.
    • Parameter: domainname, for example themeforest.net
    • Return: Array with 3 keys, google, bing, yahoo
    • Code example:
      $result = $DomainTools->IndexedPages("themeforest.net");
echo $result['google'] . "<br />";
echo $result['bing'] . "<br />";
echo $result['yahoo'] . "<br />";
  6. Backlinks: Get the number of backlinks for a domainname from Google and Yahoo.
    • Parameter: domainname, for example themeforest.net
    • Return: Array with 2 keys, google, yahoo
    • Code example:
      $result = $DomainTools->Backlinks("themeforest.net");
echo $result['google'] . "<br />";
echo $result['yahoo'] . "<br
